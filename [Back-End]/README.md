# Back-End Files

This folder contains all back-end/server-side code for IGNIS.

## Contents

- **api/** - Next.js API routes
  - `analyze/route.ts` - GPT-4 analysis endpoint

## Note

These files are also present in the `app/api/` directory for Next.js API routing. The `app/api/` directory is the working directory that Next.js uses at runtime.
